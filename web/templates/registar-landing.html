Star Relay

A public relay server for Star. Copy the URL below into the Star client to play games over the internet with friends.

Getting Started

Host a Game

  1. Download Star and run it
  2. Enter the server URL above, click Host a Game
  3. Select your game and click Start Hosting
  4. Share the invite code (XXXX-XXXX) with your friend

Join a Game

  1. Download Star and run it
  2. Enter the server URL above, click Join a Game
  3. Enter the invite code and click Connect
  4. Use the displayed address in your game client

How It Works

Star first attempts a direct P2P connection between host and peer (UDP hole punching). If that fails, game traffic is relayed through this server. All P2P traffic is encrypted with mutual TLS using per-session certificates. All communication uses HTTP/3 (QUIC).
